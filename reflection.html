Reflection

Reflecting on stuff...

It was not the first time we encountered html, so the experience was not relatively new to us.

Although, there were new functions that we encountered, such as calendar, table, textbox and others. For us it was something new and is very useful that we will use in the future. It was very informative in a sense that the speaker emphazises that "if you think somebody might have needed that there is probably a function for it". We belive that this is something we unconciously do and makes our coding life more stressful than it already is.

It was our first time learning bootstrap and we believe that it is very useful and amazing. :D Now that the mobile age is improving the bootstrap template is almost a must to be able to adjust to the new generation. This also makes developing web easier and more convinent.

Through this workshop, we were able to approach web developing with more courage, unlike before, we were afraid of doing a website because we have no idea what it is and how to do it. We were also amazed by the things we were able to learn and we hope to learn more about web development. Overall, it was a good and helpful experience.
